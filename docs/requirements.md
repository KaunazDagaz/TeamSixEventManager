### Project Domain Description

**EventManager** is a service-aggregator platform designed to simplify how people discover and organize events. The core idea is to provide organizers with the ability to create dedicated pages for their events - such as concerts or festivals - while giving users a centralized hub to find these activities and express interest through likes and comments.

The platform addresses the fragmentation of the event market, where users often have to check multiple resources to find different types of gatherings, like sports matches or concerts. By using this single resource, users can access all events in one place while filtering out content that does not interest them.

Key functional aspects include:
* **Interactive Engagement**: Users can mark their desire to visit an event, leave likes, and post comments.
* **Personalized Management**: A personal cabinet allows users to track events they want to visit, see liked events, and view a personal event calendar.
* **Organizer Tools**: Organizers can create and edit event information and access detailed analytics, such as interest levels and activity graphs.
* **Tiered Access**: A robust role system—ranging from Unauthorized Users to Moderators - ensures secure interaction and content quality.

---

### Target Audience

The project is designed for two primary groups:

* **Event Attendees**: Anyone interested in participating in various events who wants a convenient, centralized way to find and plan their activities.
* **Organizers**: Individuals or entities who need a professional platform to host all their events in a standardized format and reach a wider audience.
